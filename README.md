# 🏨 Hotel Booking Website

A full-stack MERN (MongoDB, Express, React, Node.js) **Hotel Booking Web Application** that allows users to explore rooms, view details, book rooms, and manage their profiles. Built with responsiveness, performance, and elegant design in mind.

---

## ✨ Features

- 🔐 User Signup / Login / Profile Management
- 📆 Room Booking System with Form Submission
- 🛏️ Detailed Room Pages with Favorite Option
- 💬 Contact Page with Backend Integration
- 📊 Dashboard UI with Statistics & Booking Info
- 🌙 Dark/Light Theme Toggle
- 📷 Dynamic Image Gallery
- 📱 Fully Responsive (Mobile & Desktop)

---

## 🧑‍💻 Tech Stack

| Frontend        | Backend         | Database   |
|----------------|------------------|------------|
| React, TailwindCSS | Node.js, Express | MongoDB    |
